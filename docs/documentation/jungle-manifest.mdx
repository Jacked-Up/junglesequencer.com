The Jungle Manifest is a file used to store metadata about the tree.

The manifest is automatically created when a new tree is added to the project.
The manifest is not built to the application and is only available in the Unity Editor.

---
## Code Reference

:::warning WARNING
The manifest is not built to the application and is only available in the Unity Editor.
Ensure you are only calling these methods from an editor script.
:::
